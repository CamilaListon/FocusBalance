# 📱 FocusBalance

FocusBalance é uma aplicação web focada em bem-estar digital e produtividade, projetada para ajudar os usuários a monitorarem e gerenciarem o tempo gasto em aplicativos e telas no dia a dia. Com uma interface minimalista e visualmente calmante baseada em tons de verde, o sistema incentiva o equilíbrio saudável entre tecnologia e vida real.

---

## 🚀 Funcionalidades Chave

* **🔐 Autenticação Segura:** Sistema de Login e Cadastro de usuários utilizando Axios e validações em tempo real.
* **📊 Painel de Estatísticas (Dashboard):**
    * **Streak Atual:** Contador de dias consecutivos monitorando o tempo de tela para incentivar a consistência.
    * **Média Diária:** Cálculo automático do tempo de uso.
    * **Destaque inteligente:** Exibição do aplicativo mais utilizado.
* **➕ Registro Rápido:** Formulário inline e responsivo para adicionar novos registros de aplicativos e minutos gastos.
* **📅 Histórico Gerenciável:** Tabela interativa com scroll responsivo onde é possível visualizar, editar (✏️) ou excluir (🗑️) o tempo de aplicativos já registrados.
* **⚙️ Perfil Personalizado:** Alteração de dados cadastrais, troca de senha e inclusão de URL de imagem de perfil (avatar), que é integrada dinamicamente ao cabeçalho da dashboard.
* **🚨 Zona de Perigo:** Área isolada e segura para a exclusão definitiva da conta e do histórico.

---

## 🛠️ Tecnologias Utilizadas

O ecossistema do projeto foi construído utilizando as seguintes tecnologias no ecossistema JavaScript:

* **React** (com Vite para um ecossistema de desenvolvimento rápido)
* **React Router DOM** (Gerenciamento de rotas e navegação SPA)
* **Axios** (Consumo de API HTTP de forma limpa e assíncrona)
* **CSS3 Puro** (Arquitetura modular de estilos com variáveis globais `:root` focadas no conceito *Clean & Calm UI*)
